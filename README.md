# B站弹幕 XML 转 ASS 字幕工具

将 B 站（哔哩哔哩）下载的弹幕 XML 文件批量转换为 ASS 字幕格式，方便在主流播放器（PotPlayer、MPV、VLC 等）中加载观看。

## 功能特性

- 批量扫描指定目录下的所有 `.xml` 弹幕文件
- 逐个转换为同名 `.ass` 字幕文件
- 自动将原始 XML 文件移动到 `XML_Backup` 子目录，避免重复处理
- 支持滚动弹幕（从右向左）、顶部固定、底部固定等多种 B 站弹幕类型
- 保留原始颜色、字号等样式信息
- 支持命令行参数传入路径，也可交互式输入

## 使用方法

### 方式一：命令行参数

```bash
python "B站弹幕XML转ASS.py" "D:\downloads\danmaku"
```

### 方式二：双击 / 交互式运行

直接运行脚本，按提示输入包含 XML 文件的目录路径即可。

## 弹幕类型映射

| B 站 mode | 类型 | ASS 渲染方式 |
|-----------|------|--------------|
| 1, 2, 3   | 滚动弹幕 | `\move` 从屏幕右侧滚动到左侧 |
| 4         | 底部弹幕 | `\pos` 固定在屏幕底部 |
| 5         | 顶部弹幕 | `\pos` 固定在屏幕顶部 |
| 其它      | 未知模式 | 按滚动弹幕处理 |

## 可调参数

在脚本顶部可根据需要修改以下渲染参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `PLAY_RES_X` | 1920 | 画面宽度 |
| `PLAY_RES_Y` | 1080 | 画面高度 |
| `SCROLL_SPEED` | 150.0 | 滚动弹幕速度（像素/秒），值越小越慢 |
| `STATIC_DURATION` | 5.0 | 固定弹幕停留秒数 |
| `FONT_SIZE` | 38 | 默认字号 |
| `ROW_HEIGHT` | 56 | 滚动弹幕行高 |
| `TOP_MARGIN` | 30 | 顶部边距 |
| `BOTTOM_MARGIN` | 80 | 底部边距（避开播放器进度条） |
| `FONT_NAME` | Microsoft YaHei | 字体名称 |

## 注意事项

- 输出 ASS 文件使用 UTF-8 with BOM 编码，兼容 Windows 记事本等编辑器
- 同一目录下若存在重名备份 XML，会自动重命名为 `文件名_1.xml`、`文件名_2.xml` 等
- 若 XML 文件解析失败或弹幕列表为空，对应文件不会生成 ASS 字幕
- 建议在播放器中将字幕字体设置为「微软雅黑」以获得最佳显示效果

## 依赖

仅使用 Python 标准库（`os`、`sys`、`shutil`、`xml.etree.ElementTree`），无需额外安装第三方包。

Python 3.6+ 即可运行。
